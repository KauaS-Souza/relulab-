# relulab - Biolab Farmacêutica Landing Page

## 📃 Descrição do Projeto
O **relulab** é uma landing page institucional moderna desenvolvida para a **Biolab Farmacêutica**, alinhada ao posicionamento de marca e ao slogan institucional **"Viva a Evolução!"**. O projeto visa renovar a presença digital da empresa, utilizando como base estrutural e arquitetura visual o padrão de componentes do **Relume Figma Kit**.

## 🛠️ Tecnologias Utilizadas
* **HTML5**: Estruturação semântica e acessível.
* **CSS3**: Estilização responsiva, variáveis de escopo e Grid/Flexbox layouts.
* **JavaScript (Vanilla)**: Interações de interface e transições de navegação.
* **Design Pattern**: Baseado nos componentes do Relume Figma Kit.
